Time Serious

Time Serious is an editors lab hackathon project which takes a data input, analyses it, and outputs a news report in natural language, with publication-ready graphics. Read more about the project at the GitHub repo, or check out the demo below.

Currently it only works with a google spreadsheet formatted like the example emissions data.
